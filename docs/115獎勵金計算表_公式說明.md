# 115 獎勵金計算表 各格公式說明

## 第一區：獎勵金計算（Row 4-6）

### 門診醫療（Row 4）

| 格位 | 名稱 | 公式 | 說明 |
|---|---|---|---|
| B4 | 掛帳金額 | 手動輸入 | 門診健保掛帳金額 |
| C4 | 醫療折讓 | `=ROUND(B4*D8, 0)` | 掛帳 × 折讓率 |
| D4 | 健保不給付、部分負擔 | 手動輸入 | 健保不給付 + 部分負擔及其他款項 |
| E4 | 健保申復/剔除 | `=F24-C16` | 本年度保險申復小計 − 本年度無法沖帳小計 |
| F4 | 支出部份 | 手動輸入 | 510601 門診醫療成本 |
| G4 | 獎前收支淨餘數 | `=B4-C4+D4+E4-F4` | 掛帳 − 折讓 + 不給付 + 剔除 − 支出 |
| H4 | 醫療作業基金 5% | `=ROUND(G4*0.05, 0)` | 淨餘 × 5% |
| I4 | 繳局統籌款 | `=ROUND(G4*0.95*0.1, 0)` | 淨餘 × 95% × 10% |
| J4 | 醫師獎勵金 | `=G4-H4-I4-K4` | 餘數（淨餘 − 基金 − 繳局 − 員工） |
| K4 | 員工獎勵金 | `=ROUND(G4*0.95*0.9*0.3, 0)` | 淨餘 × 95% × 90% × 30% |

### 衛生保健（Row 5）

| 格位 | 名稱 | 公式 | 說明 |
|---|---|---|---|
| B5 | 掛帳金額 | 手動輸入 | 衛生保健（行動醫院）掛帳金額 |
| C5 | 醫療折讓 | `=C6-C4` | 小計折讓 − 門診折讓（差額分攤） |
| D5 | 健保不給付 | 手動輸入 | （目前為空） |
| E5 | 健保申復/剔除 | 手動輸入 | （目前為空） |
| F5 | 支出部份 | 手動輸入 | 衛生保健支出（行動醫院動支單合計） |
| G5 | 獎前收支淨餘數 | `=B5-C5+D5+E5-F5` | 同門診公式 |
| H5 | 醫療作業基金 5% | `=ROUND(G5*0.05, 0)` | 同門診 |
| I5 | 繳局統籌款 | `=ROUND(G5*0.95*0.1-B8, 0)` | 淨餘 × 95% × 10% **− 衛生局加班費** |
| J5 | 醫師獎勵金 | `=G5-H5-I5-B8-K5` | 餘數 − 加班費（加班費與 I5 中的負數互抵，醫師實際不受影響） |
| K5 | 員工獎勵金 | `=ROUND(G5*0.95*0.9*0.7, 0)` | 淨餘 × 95% × 90% × **70%**（非 30%） |

### 門診 vs 衛保 關鍵差異

| 項目 | 門診醫療 | 衛生保健 |
|---|---|---|
| 員工比例 | 0.3（27%） | **0.7（63%）** |
| 繳局扣加班費 | 否 | **是（−B8）** |
| 醫師公式 | `=G4-H4-I4-K4` | `=G5-H5-I5-B8-K5` |

| C6 | 折讓合計 | `=ROUND(B6*D8, 0)` | 總掛帳 × 折讓率 |
| D6 | 不給付合計 | `=SUM(D4:D5)` | |

| G6 | 淨餘合計 | `=B6-C6+D6+E6-F6` | |
| H6 | 基金合計 | `=SUM(H4:H5)` | |
| I6 | 繳局合計 | `=SUM(I4:I5)` | |

## 公式邏輯摘要

```
獎前淨餘 = 掛帳 − 折讓 + 不給付 + 剔除沖帳 − 支出

基金 5%     = ROUND(淨餘 × 0.05)
繳局統籌款  = ROUND(淨餘 × 0.95 × 0.1)         ← 門診
繳局統籌款  = ROUND(淨餘 × 0.95 × 0.1 − 加班費) ← 衛保
員工獎勵金  = ROUND(淨餘 × 0.95 × 0.9 × 0.3)   ← 門診
員工獎勵金  = ROUND(淨餘 × 0.95 × 0.9 × 0.7)   ← 衛保
醫師獎勵金  = 淨餘 − 基金 − 繳局 − 員工         ← 餘數（門診）
醫師獎勵金  = 淨餘 − 基金 − 繳局 − 加班費 − 員工 ← 餘數（衛保）

115年起：繳局 + 醫師 + 員工 全部掛 210205/01 繳局統籌款
```
