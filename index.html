⚙️ 連線設定

在 GitHub 建立一個名為 records.json 的 Gist，
內容填 []，把網址最後一段 ID 貼到這裡。
GitHub → Settings → Developer settings → Personal access tokens
建立 token，勾選 gist 權限即可。只需在本台電腦設定一次，
其他電腦各自設定自己的 token（Gist ID 相同）。
